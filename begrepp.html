Begrepp

Begrepp

HTTP

Hypertext Transfer Protocol - protokoll för överföring av webbdata.

OSI

Open Systems Interconnection - en modell för hur nätverk kommunicerar i sju lager.

ITU

International Telecommunication Union - FN-organ för telekommunikation och radiofrekvensspektrum.

W3C

World Wide Web Consortium - organisation som utvecklar standarder för webben.

RJ-45

Registered Jack 45 - kontakt för nätverkskablar (Ethernet).

USB

Universal Serial Bus - standard för datakommunikation och strömförsörjning mellan enheter.

IEEE

Institute of Electrical and Electronics Engineers - organisation som utvecklar teknikstandarder.

TCP/IP

Transmission Control Protocol/Internet Protocol - grundläggande protokoll för internetkommunikation.

IETF

Internet Engineering Task Force - organisation som utvecklar internetstandarder.

Fysisk Adress

MAC-adress som identifierar enhet på en lokal nätverk.

Frames

Datapaket på datakopplingen (lager 2) i OSI-modellen.

802.?

IEEE 802-serien - standarder för lokala nätverks- och metropolitana nätverk (LAN/MAN).

Broadcast

Sändning av data från en källa till alla mottagare på ett nätverk.

Router

Nätverksenhet som dirigerar datapaket mellan olika nätverk.

Switch

Nätverksenhet som förbinder enheter på samma nätverk och dirigerar data baserat på MAC-adresser.

Access Punkt

Trådlös nätverksenhet som möjliggör trådlös anslutning till ett nätverk.

DNS

Domain Name System - system för att översätta domännamn till IP-adresser.

FTP

File Transfer Protocol - protokoll för filöverföring mellan datorer.

Peer to Peer

Nätverkstopologi där datorer är jämlika och kan kommunicera direkt med varandra.

Logisk Topologi

Hur data flödar i ett nätverk, oavsett fysisk konfiguration.

Fysisk Topologi

Den faktiska fysiska anordningen av nätverksenheterna och kablarna.

Parallell Överföring

Överföring av flera bitar samtidigt genom flera ledningar.

Seriell Överföring

Överföring av data en bit åt gången genom en enda ledning.

Synkron

Datakommunikation där sändare och mottagare är synkroniserade med samma klocka.

Asynkron

Datakommunikation där sändare och mottagare inte behöver vara synkroniserade.

Duplex

Möjligheten att skicka och ta emot data samtidigt (full duplex) eller växelvis (half duplex).

MAC-adress

Media Access Control-adress - fysisk adress som identifierar en enhet på ett lokalt nätverk.

IP-adress

Internet Protocol-adress - logisk adress som identifierar en enhet på internet.

ARP

Address Resolution Protocol - protokoll för att översätta IP-adresser till MAC-adresser.

Frame

Dataenhet på datakopplingslager (lager 2) som innehåller MAC-adresser.

PDU

Protocol Data Unit - dataenhet på en viss nivå i OSI-modellen.

UDP

User Datagram Protocol - transportprotokoll som är snabbare men mindre tillförlitlig än TCP.

Port

Nummer som identifierar en specifik tjänst eller process på en dator.

Socket

Kombination av IP-adress och port som representerar en ände av en nätverksanslutning.

3-Way Handshake

Tre-stegs process för att etablera en TCP-anslutning mellan två datorer.
