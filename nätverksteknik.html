Nätverksteknik

Nätverksterminologi

HTTP

Hypertext Transfer Protocol - protokollet för att överföra webbsidor.

OSI

Open System Interconnection - en modell för hur nätverk kommunicerar.

ITU

Internationella Teleunionen - en organisation för telekommunikation.

W3C

World Wide Web Consortium - organisation som sätter standarder för webben.

RJ-45

Register Jack-45 - en standard för nätverkskontakter.

USB

Universal Serial Bus - standard för datöverföring och strömförsörjning.

IEEE

Institute of Electrical and Electronics Engineers - organisation för teknikstandarder.

TCP/IP

Transmission Control Protocol/Internet Protocol - grundläggande internetprotokoll.

IETF

Internet Engineering Task Force - utvecklar internetstandarder.

Fysisk Adress (MAC-adress)

En unik identifierare för nätverkskort som finns på varje enhet.

Frames

Strukturen för datablock som skickas över nätverk.

802.?

802.11 är standard för trådlöst internet, 802.3 för nätverk med tråd.

Broadcast

Bredsändning - skicka data till alla enheter i ett nätverk.

Router

En enhet som distribuerar fibernätet via radiovågor eller nätverkskabel för att möjliggöra trådlös eller trådbunden internetanslutning.

Switch

Sammankopplar flera enheter samtidigt inom ett lokalt nätverk.

Access Point

Åtkomstpunkt - möjliggör trådlös anslutning till ett nätverk.
