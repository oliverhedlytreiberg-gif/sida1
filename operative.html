Operativsystem

Viktiga operativsystem genom historien

UNIX (1969)

UNIX utvecklades vid Bell Labs av bland andra Ken Thompson och Dennis Ritchie. Det var ett av de första operativsystemen som skrevs i ett högnivåspråk (C), vilket gjorde det portabelt. UNIX har haft mycket stor betydelse och ligger till grund för flera senare system, såsom Linux och macOS.

MS-DOS (1981)

MS-DOS (Microsoft Disk Operating System) utvecklades av Microsoft och användes främst på persondatorer från IBM. Det var textbaserat och krävde att användaren skrev kommandon. MS-DOS lade grunden för Microsofts senare operativsystem.

Microsoft Windows (1985)

Windows började som ett grafiskt gränssnitt ovanpå MS-DOS. Med tiden utvecklades det till ett fullständigt operativsystem. Windows är idag det mest använda operativsystemet för persondatorer och finns i många versioner, exempelvis Windows 95, XP, 7, 10 och 11.

Classic Mac OS / macOS (1984 / 2001)

Apple lanserade Macintosh med Classic Mac OS 1984, ett av de första operativsystemen med grafiskt användargränssnitt för konsumenter. År 2001 ersattes det av Mac OS X (senare macOS), som är UNIX-baserat och används på Apples datorer än idag.

Linux (1991)

Linux skapades av Linus Torvalds som ett fritt och öppet operativsystem. Det används i många olika varianter (distributioner), till exempel Ubuntu, Debian och Fedora. Linux är mycket vanligt på servrar, i superdatorer och inbyggda system.

Android (2008)

Android är ett Linux-baserat operativsystem utvecklat av Google, främst för smartphones och surfplattor. Det är idag världens mest använda mobila operativsystem och används av många olika tillverkare.

iOS (2007)

iOS utvecklades av Apple för iPhone och bygger på samma tekniska grund som macOS. Systemet är känt för hög säkerhet och stark integration med Apples övriga produkter.
